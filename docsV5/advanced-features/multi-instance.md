# 多开管理

得益于AUTO-MAS强大的模拟器管理，在使用常见模拟器时无需配置繁杂的启动指令。你只需要在脚本中直接选择脚本需要启动的模拟器实例即可。

专项适配的脚本会自动处理模拟器的路径和adb路径，无视脚本内的相关配置。
