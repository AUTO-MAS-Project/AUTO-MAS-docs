## 参考信息

### 常见日期时间格式符号对照表

| 符号     | 表示内容         | 示例       |
|----------|-----------------|------------|
| `%Y`     | 四位数的年份     | 2025       |
| `%m`     | 两位数的月份     | 07         |
| `%d`     | 两位数的日期     | 16         |
| `%H`     | 小时（24小时制） | 14         |
| `%M`     | 分钟            | 30         |
| `%S`     | 秒              | 45         |
| `%f`     | 毫秒（3位）	    | 123        |
| `%f`     | 微秒（6位）	    | 123456     |
| `%A`     | 星期全称         | Wednesday  |
| `%a`     | 星期简称         | Wed        |
| `%B`     | 月份全称         | July       |
| `%b`     | 月份简称         | Jul        |
