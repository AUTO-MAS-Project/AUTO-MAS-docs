# AUTO-MAS 开源项目信息公开说明

为保证 AUTO-MAS 项目与相关社区的持续健康发展，现将以下内容公示

## 项目收支

众所周知，开源项目不可能长期依靠开发者用爱发电，为保证项目的持续发展，激励开发者参与项目开发，我们通过多种渠道接受用户捐赠，获取各类收益。感谢各位用户一直以来的理解与宽容，但不透明的项目资金运营模式，可能造成部分用户的困扰，故正式公布本项目的资金使用原则与收支表。

- **宣发收益与项目收益独立**：所有项目宣传者在项目宣传中获得的直接收益（包括但不限于：广告商单、小店商品分红、创作者激励、充电收益）均归宣传者个人所有，项目组直接出资赞助的宣传活动除外。
- **开发者收益与项目收益独立**：所有项目开发者在项目开发过程中获得的直接收益（包括但不限于：用户定向捐赠）均归开发者所有。
- **账单结算后录入**：所有项目资金变动情况，在确认完成结算后录入并公开，由于开发者忙绿，部分条目可能会延迟登记，或多条合并登记。
- **使用实物激励项目开发者**：为避免项目开发者关于激励金额的不必要的争议，本项目不采用现金分红方式进行激励，采用统一购置实物礼品的方式回馈项目开发者，购置与分配方案由项目作者制定。
- **项目收益全部用于项目**：本项目所有收益，必须全部用于项目相关事务，未被项目收支表记录的内容将被视为无效。

## 隐私政策

为更好地提供服务，AUTO-MAS 将自动收集以下信息：

- 软件版本号
- 运行时错误信息

AUTO-MAS 尊重并保护用户隐私，上报的所有信息均经过匿名化处理，不包含任何个人身份信息。所收集的数据存储于 AUTO-MAS 官方服务器，不会传输至任何第三方机构或设施。
